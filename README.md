# Perun Programming Language

The Perun C++ compiler is __under heavy development__ as it is being rewritten from a prototype.

## Building

### Dependencies

* CMake >= 3.0
* C++14-compatible compiler (such as GCC >= 5.0 or Clang >= 3.6)

### Instructions

#### Linux

```
mkdir build
cd build
cmake ..
make
cd ..
build/perun $YOUR_FILE.per
```
